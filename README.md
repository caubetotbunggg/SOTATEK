# SOTATEK — ZET Template Matching Detector

Zero-shot pattern detection on technical drawings: upload a **pattern** (template) and a **drawing**, run inference online, and view bounding boxes on the result image.

## How to use

1. Pick one of the **Quick test examples** below (or upload your own pattern + drawing).
2. Adjust detection parameters if needed (scale range matters most).
3. Click **Detect** (examples auto-run when selected).
4. Review the visualization with drawn bounding boxes and the JSON output.
